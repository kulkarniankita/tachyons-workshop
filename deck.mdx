---

## BEM

```html
<div className="Card">
  <div className="Card__Media" />
  <img className="Card__Avatar" />
  <h3 className="Card__Category">
    Arts & Crafts
  </h3>
  <h2 className="Card__Title">
    Wooden Materials and Sculpting Machinery
  </h2>
  <div className="Card__Count">22</div>
</div>
```

---

## CSS Modules
```jsx
import styles from './CourseCard.css';

const CourseCard = () => (
  <div className={styles.Card}>
    <div className={styles.Media} />
    <img className={styles.Avatar} />
    <h3 className={styles.Category}>
      Arts & Crafts
    </h3>
    <h2 className={styles.Title}>
      Wooden Materials and Sculpting Machinery
    </h2>
    <div className={styles.Count}>22</div>
  </div>
);
```

## Components With Tachyons + Angular

### Styled Directives

```js
// Heading
@Directive({
  selector: '[appHeading]',
  host: { class: 'helvetica-neue black lh-title' },
})
export class HeadingDirective {}

// Text
@Directive({
  selector: '[appText]',
  host: { class: 'helvetica-neue black lh-copy' },
})
export class TextDirective {}
```

---

```html
<a appBlockLink class="mb2" href="#">

```
